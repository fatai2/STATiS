Selection Tool for Appropriate Tests in Statistics (STATiS)

This interface will assist you to choose the appropriate test for your statistical analysis using the information you provide by clicking the options below.

  • What type of analysis to you want to perform?
  • I want to measure associations between variables
    • What type of association do you want to perform?
    • I want to measure the association between scores derived from two or more raters/instruments — agreement
      • What is the level of measurement for these scores?
      • The scores are nominal

        • The test you need is Kappa

      • The scores are ordinal
        • How many raters/instruments were used to derive the scores?
        • Scores were measured by only 2 raters/instruments

          • The test you need is Kendall's tau

        • Scores were measured by 3 or more raters

          • The test you need is Kendall's W

      • The scores are interval
        • Are the variables normally distributed?
        • Yes, they meet criteria for parametric analysis

          • The test you need is Pearson's r

        • No, they do not meet criteria for parametric analysis

          • The test you need is Spearman's rho

    • I want to measure the relationship between two variables — Correlation
      • What is the level of measurement for the base variable?
      • One of the variables is binary (dichotomous)
        • What is the level of measurement of the second variable?
        • The second variable is nominal
          • How many levels does the second variable have?
          • The second variable is binary (dichotomous)

            • The test you need is Phi

          • The second variable has 3 or more levels

            • The test you need is Chi-square

        • The second variable is ordinal

          • The test you need is Rank Biserial

        • The second variable is interval

          • The test you need is Point Biserial

      • One of the variables is ordinal

        • The test you need is Spearman's rho

      • One of the variables is interval
        • Are the variables normally distributed?
        • Yes, the variables meet criteria for parametric analysis

          • The test you need is Pearson's r

        • No, the variables do not meet criteria for parametric analysis

          • The test you need is Spearman's rho

    • I want to measure the relationship between predictor and outcome variables⁠ — Regression
      • What is the level of measurement for the outcome variable?
      • The outcome variable is interval

        • The test you need is Linear regression

      • The outcome variable is not interval
        • How many levels does the outcome variable have?
        • The outcome variable is binary

          • The test you need is binary (binomial) logistic regression

        • The outcome variable has 3 or more ordinal levels

          • The test you need is ordinal logistic regression

        • The outcome variable has 3 or more nominal levels

          • The test you need is nominal logistic regression

  • I want to measure differences between/among groups
    • How many groups are you comparing?
    • I am comparing one group to a specified value
      • What is the level of measurement for the variable?
      • The variable of interest is nominal

        • The test you need is Binomial test

          or One-sample chi-square

      • The variable of interest is ordinal

        • The test you need is Runs test

      • The variable of interest is interval

        • The test you need is One-sample t-test

    • I want to measure differences between two groups
      • Are the samples related/paired?
      • No, the samples are not paired nor has the measurement been repeated
        • What is the level of measurement of the variable?
        • The variable of interest is nominal
          • What is the sample size of the data?
          • The sample size is less than 20

            • The test you need is Fisher's exact

          • The sample size is 20 or more

            • The test you need is Chi-square

        • The variable of interest is ordinal

          • The test you need is Mann-Whitney U

            or Wilcoxon Rank Sum

        • The variable of interest is interval
          • Are the variables normally distributed?
          • Yes, the variables meet criteria for parametric analysis

            • The test you need is Independent samples t-test

          • No, the variables do not meet criteria for parametric analysis

            • The test you need is Mann-Whitney U

              or Wilcoxon Rank Sum

      • Yes, the samples are paired (or the measurement has been repeated)
        • What is the level of measurement of the variable?
        • The variable of interest is nominal

          • The test you need is McNemar's Chi-square

        • The variable of interest is ordinal

          • The test you need is Wilcoxon Signed Ranks

        • The variable of interest is interval
          • Are the variables normally distributed?
          • Yes, the variables meet criteria for parametric analysis

            • PThe test you need is Paired samples t-test

          • No, the variables do not meet criteria for parametric analysis

            • The test you need is Mann-Whitney U

              or Wilcoxon Rank Sum

    • I want to measure differences among three or more groups
      • Are the samples related/matched?
      • No, the samples are not matched nor has the measurement been repeated
        • What is the level of measurement of the variable?
        • The variable of interest is nominal

          • The test you need is Chi-square

        • The variable of interest is ordinal

          • The test you need is Kruskal-Wallis One-Way Analysis of Variance

        • The variable of interest is interval
          • Are the variables normally distributed?
          • Yes, the variables meet criteria for parametric analysis

            • The test you need is Independent samples t-test

          • No, the variables do not meet criteria for parametric analysis

            • The test you need is Analysis of Variance

      • Yes, the samples are paired (or the measurement has been repeated)
        • What is the level of measurement of the variable?
        • The variable of interest is nominal

          • The test you need is Cochran's Q

        • The variable of interest is ordinal

          • The test you need is Friedman's Test (Two-Way Analysis of Variance by Ranks)

        • The variable of interest is interval
          • Are the variables normally distributed?
          • Yes, the variables meet criteria for parametric analysis

            • The test you need is Repeated measures Analysis of Variance

          • No, the variables do not meet criteria for parametric analysis

            • The test you need is Friedman's Test (Two-Way Analysis of Variance by Ranks)
